# 📌 PostFlow

Simple app for displaying posts and comments from an API

---

## 🚀 Demo

https://nastia-pirogova.github.io/PostFlow/

---

## ✨ Features

- Load posts from API
- Show comments (accordion style)
- Delete posts from UI
- Open post in modal
- Responsive layout

---

## 🧠 Tech Stack

- JavaScript (ES6+)
- Vite
- JSONPlaceholder API
- HTML5 / CSS3
